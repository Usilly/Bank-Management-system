# 🏦 Bank Management System

A desktop-based Bank Management System built using Python and CustomTkinter.

## Features

- User Registration
- Secure Login
- Deposit Money
- Withdraw Money
- Balance Enquiry
- Delete Account
- Random Account Number Generation
- JSON-based Data Storage

## Technologies Used

- Python
- CustomTkinter
- JSON
- Object-Oriented Programming (OOP)

## Project Structure

```
main.py
data.json
requirements.txt
```
